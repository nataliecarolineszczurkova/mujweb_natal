První velký nadpis

Druhý nadpis, veliost h2, tj. menší než h1

Tagy jsou značky, které určují, jak bude text, který píšu ve VSCode vypadat! Např. tag "p" říká, že na webové stránce se objeví odstavec textu, ten odstavec se snaží přizpůsobit prohlížeči. Na telefonu a stejně tak na pc, bych měl vidět to samé

Ve VSCode, budeme editovat naší webovou stránku, pomocí Live Server hned uvidíme, jak ten web vypadá. Updatovat a posílat změny na Github budeme jen, když si řeknee. Náhle v live Server vidím jen já, jakmile to pošlu na Github, tak to vidí celý svět.
